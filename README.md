# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with vanilla HTML, CSS, and JavaScript.

## Demo

🎮 **[Play the game here](https://ollie202.github.io/Rock-Paper-Scissors-Game/)**

## Features

- Play Rock, Paper, or Scissors against the computer
- Computer picks its move randomly and fairly (equal 1/3 probability each)
- Tracks your Wins, Losses, and Ties across sessions using `localStorage`
- Reset your score at any time with the Reset button
- Dark theme UI

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Ollie202/Rock-Paper-Scissors-Game.git
   ```
2. Open `index.html` in your browser.

No build step or dependencies required.

## Built With

- HTML
- CSS
- JavaScript (vanilla)
- Browser `localStorage` for score persistence
